# AvitoMiniApp

Учебный проект настольного приложения на C# (WPF), реализующий упрощённый аналог Avito.

## Функционал

- Просмотр списка объявлений с фильтрацией и поиском.
- Авторизация пользователей (логин/пароль).
- Личный кабинет: добавление, редактирование и удаление объявлений.
- Статусы объявлений: *Активно* / *Завершено*.
- Фиксация прибыли по завершённым объявлениям и вывод статистики.

## Технологии

- C# .NET (WPF, MVVM)
- SQLite / MSSQL
- Entity Framework Core
- Git, GitHub (Issues, Projects, Wiki, Actions)

## Цель проекта

Освоение разработки настольного приложения и ведения проекта в GitHub с использованием веток, Pull Request, Kanban-доски, релизов и CI/CD.
